# Release v1.4.0 - Personnalisation & Multilangue

## Informations generales

- **Titre :** `Personnalisation & Multilangue`
- **Tag :** `v1.4.0`
- **Date :** 17 fevrier 2026
- **Statut :** Stable / En cours (multilangue)

## Description

La version **v1.4.0** enrichit l'experience utilisateur de Pepites Academy en offrant davantage de controle sur l'apparence et le comportement de l'application. Elle introduit egalement les premieres briques du support multilangue, qui sera finalise dans une prochaine mise a jour.

## Nouveautes majeures

- **Choix du theme :**
  - Vous pouvez desormais choisir entre un theme clair, un theme sombre ou laisser l'application suivre automatiquement le reglage de votre appareil.
  - Votre preference est sauvegardee et appliquee immediatement.
  - Accessible depuis Reglages > Theme.

- **Modification du profil academicien :**
  - Il est maintenant possible de modifier les informations d'un academicien directement depuis sa fiche de profil.
  - Toutes les donnees (identite, scolarite, football) sont editables.

- **Page A propos :**
  - Une nouvelle page A propos est disponible dans les reglages, avec le logo de l'application et les informations de version.

- **Reglages des notifications :**
  - Une page dediee permet de consulter et gerer les preferences de notifications.

- **Changement de langue (en cours) :**
  - Une nouvelle option permet de basculer entre le francais et l'anglais depuis les reglages.
  - Le changement de langue est immediat et sauvegarde pour les prochaines sessions.
  - **Important :** les textes de l'application ne sont pas encore tous traduits. La traduction complete de l'ensemble des ecrans sera livree dans une prochaine version. Pour le moment, seule la structure technique est en place.

## Ameliorations & Corrections

- L'orientation de l'ecran est desormais verrouillee en mode portrait pour une meilleure stabilite d'affichage.
- Les boutons "Mes annotations" et "Mes ateliers" du tableau de bord encadreur sont maintenant fonctionnels.
- Les menus Notifications et A propos sont connectes dans les reglages admin et encadreur.
- Le menu Sauvegarde a ete retire des reglages administrateur (fonctionnalite geree automatiquement).

---

_Note : Cette version pose les fondations du support multilangue de l'application. La traduction integrale de tous les ecrans en anglais sera finalisee dans la version suivante. D'ici la, l'application reste entierement utilisable en francais._
